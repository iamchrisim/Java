
1.Import Project directory to eclipse
2.Right click Project, then go to properties
3.Remove the unbounded JRE (JRE System Library with (unbounded))
4.Add current JRE System Library that you are using. Press Add Library and install new one.
5.Press Ok button.
6.There will be still errors on the programs, so press the player.java and then you can change the version to JRE 1.5
